熟悉实验环境

一、实验目的：
（1）了解Visual C++ 6.0的特点；
（2）熟悉Visual C++ 6.0的开发环境；
（3）学习用Visual C++ 6.0编写标准的C++控制台程序。

二、实验内容
使用Visual C++ 6.0建立一个非图形化的标准C++程序，编译、运行以下程序：
#include <iostream>
using namespace std;
int main()
{
cout<<"Hello!\n";
cout<<"Welcome to c++!\n";
}
三、实验步骤
1． 启动Visual C++6.0
（1）从“开始”菜单中选择“程序”|Microsoft Visual Studio 6.0 |Microsoft Visual C++ 6.0，显示Visual C++ 6.0开发环境主窗口；
（2） 用鼠标左键分别单击各主菜单项，观察各下拉菜单的项目内容；
（3） 单击“文件>新建”，观察弹出的对话框；
（4） 分别单击“新建”对话框的各选项卡，查看其中的内容；
（5） 单击“结束”按钮，关闭对话框；
（6） 单击“帮助>帮助目录”，熟悉帮助的内容和使用方法。

2. 创建新项目
（1）单击“文件>新建”，
（2）在“新建”对话框中单击“工程”选项卡；在列表中选中“Win32 Console Application”；在名为“工程”的文本框中输入项目名“lab1”；在“位置”文本框中设置项目的位置；选择“创建新工作区”选项；确认在“平台”列表框中选中了“Win32”；单击“确定”按钮。
（3）选择项目类型：An empty project。
（4）单击“完成”按钮。
3. 建立C++源程序文件
（1）选择菜单命令“工程”|“添加工程”|“新建”；
（2）在“新建”对话框的“文件”选项卡中选择C++ Source File，并填入文件名“lab1”，单击“确定”按钮。
4. 编辑C++源文件内容
（1）在文件编辑窗口中输入代码；
（2）保存这个文件。
5. 建立并运行可执行程序
（1）选择菜单命令“编译”|“构建lab1.exe”，建立可执行程序；
（2）选择菜单命令“编译”|“执行lab1.exe”，运行程序，观察屏幕的显示内容。
6. 关闭工作空间

Q&A

Q：有哪几种翻译程序？
A：汇编程序，编译程序，解释程序

Q：面向过程的结构化程序设计的主要设计思路是什么？
A：自顶向下、逐步求精。采用模块分解与功能抽象，自顶向下、分而治之。

Q：面向对象程序设计的主要特点有哪些？
A：封装、继承、多态。
