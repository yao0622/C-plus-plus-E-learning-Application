UML简介

UML语言（统一建模语言）是一种可视化的的面向对象建模语言。

UML的重要内容是各种类型的图形：

软件模型的静态结构
软件模型的动态行为
模块组织和管理

UML中的9种图

类图（Class diagram）
对象图（Object diagram）
用例图（Use case diagram）
顺序图（Sequence diagram）
协作图（Collaboration diagram）
状态图（Statechart diagram）
活动图（Activity diagram）
组件图（Compomnent diagram）
实施图（Deployment diagram）

类成员表示方法

数据成员：

[访问控制属性] 名称 [重数][: 类型][-默认值][{约束特征}]

访问控制属性：
+（public），-（private），#（protected）

例如：
-hour: int
+size: area = (100, 100)

函数成员：

[访问控制属性] 名称 [(参数表)][: 返回类型][{约束特性}]

例如：
+setTime(newH: int=0, newM: int=0, newS: int=0): void

对象图

几种关系的图形标识

依赖关系：一个事物的变化可能会影响到另一个事物

图中的“类A”是源，“类B”是目标，表示“类A”使用了“类B”，或称“类A”依赖“类B” 几种关系的图形标识

作用关系??关联：两个类的对象之间相互作用

图中的“重数A”决定了类B的每个对象与类A的多少个对象发生作用，同样“重数B”决定了类A的每个对象与类B的多少个对象发生作用。

包含关系??聚集和组合

聚集表示类之间的关系是整体与部分的关系，“包含”、“组成”、“分为……部分”等都是聚集关系。

共享聚集：部分可以参加多个整体
组合：部分与整体共存，整体不存在了，部分也随之消失

继承关系??泛化

注释

在UML图形上，注释表示为带有褶角的矩形，然后用虚线连接到UML的其他元素上，它是一种用于在图中附加文字注释的机制。
