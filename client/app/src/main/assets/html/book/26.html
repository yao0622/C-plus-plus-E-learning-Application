结构体

结构体的概念

结构体是一种特殊形态的类，可以有数据成员和函数成员
与类的唯一区别是：结构体中成员的缺省访问属性为public

声明结构体

struct 结构体名称
{
  公有成员
protected:
   保护成员
private:
   私有成员
};

举例：

struct student //学生信息结构体
{
   int num; //学号
   char name[20]; //姓名
   char gender; //性别
   int age; //年龄
   float score; //成绩
   char addr[30]; //住址
}

结构变量说明

结构名 结构变量名；

注意：

结构变量的存储类型概念、它的寿命、可见性及使用范围与普通变量完全一致。
结构变量说明在结构类型声明之后，二者也可同时进行。
结构变量占内存大小可用 sizeof 运算求出：
sizeof(运算量)

结构变量的初始化和使用

初始化：说明结构变量的同时可以直接设置初值。
结构体成员的引用形式：结构变量名. 成员名

联合体

联合体是一种特殊形态的类
默认访问属性是public
联合体的全部数据成员共享同一组内存单元

声明形式：

union 联合名
{
   数据类型 成员名 1；
   数据类型 成员名 2；
   ……；
   数据类型 成员名 n；
};

联合体类型变量说明的语法形式

联合名 联合变量名；

引用形式：

联合名. 成员名

例：

union Mark
{char grade;
   bool pass;
   int percent;
};

无名联合

无名联合没有标记名，只是声明一个成员项的集合，这些成员项具有相同的内存地址，可以由成员项的名字直接访问。

例：

union
{int i;
 float f;
}

在程序中可以这样使用：
i=10;
f=2.2;
