理论目标

数据的共享与保护机制是C++的重要特性之一。

对类对象进行初始化和清理工作，是分别由构造函数和析构函数完成的。构造函数和析构函数都是由系统自动调用的。

本章的主要学习目标包括：
- 掌握C＋＋中标识符的作用域和可见性；
- 掌握变量的生存期；
- 理解静态成员的意义，掌握静态成员的定义和使用；
- 理解友元机制，掌握友元函数和友元类的定义方法；
- 掌握常引用、常对象和常成员的定义和使用；
- 了解C＋＋程序的一般组织结构，掌握常用的编译预处理命令。

实践目标

- 掌握静态成员的定义和使用；
- 掌握友元函数和友元类的定义方法；
- 掌握常引用、常对象和常成员的定义和使用；
- 掌握常用的编译预处理命令。

知识点：

（1）标识符的作用域与可见性；
（2）对象的生存期；
（3）类的静态成员；
（4）类的友元；
（5）共享数据的保护；
（6）多文件结构和和编译预处理命令。

重点：

（1）标识符的作用域与可见性；
（2）对象的生存期；
（3）类的静态成员；
（4）类的友元；

难点：

（3）常对象、常引用和常成员。
